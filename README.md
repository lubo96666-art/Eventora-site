# Eventora

Eventora е уеб приложение за разглеждане и закупуване на билети за различни събития.

Проектът е разработен със Spring Boot и следва MVC архитектура с отделни слоеве за контролери, услуги, модели и работа с данни.

## Технологии

- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- Thymeleaf
- HTML, CSS, JavaScript
- H2 Database
- Maven

## Функционалности

### Потребители

- регистрация и вход в системата
- потвърждение на профил
- възстановяване на парола
- управление на потребителски профил

### Събития

- разглеждане на събития
- търсене и филтриране
- добавяне в любими
- следене на свободни места

### Билети

- добавяне на билети в кошница
- закупуване на билети
- преглед на направени поръчки
- визуализация на електронен билет

### Допълнителни възможности

- промо кодове
- система за точки
- оценки на събития
- светла и тъмна тема

## Стартиране

Проектът се отваря в IntelliJ IDEA.

Стартира се:

```
EventoraApplication.java
```

Адрес:

```
http://localhost:8080/visualization
```

## База данни

За локална разработка се използва H2 база данни.

## Описание

Целта на приложението е да предостави система за управление на събития, потребители и продажба на билети.
