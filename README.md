# Eventora

Eventora е уеб приложение за продажба и управление на билети за събития. Приложението е изградено със Spring Boot, Thymeleaf и Spring Data JPA, като използва H2 база данни за лесно локално стартиране.

## Технологии

- Java 17
- Spring Boot 3.4.0
- Spring MVC + Thymeleaf
- Spring Data JPA
- Bean Validation
- H2 Database
- Maven

## Функционалности

- Регистрация и вход в потребителски профил
- Потвърждение с код по имейл
- Възстановяване на забравена парола
- Преглед на събития по категории
- Търсене и сортиране по дата, цена и свободни места
- Детайли за всяко събитие
- Любими събития
- Последно разгледани събития
- Светла и тъмна тема
- Кошница с промяна на броя билети
- Промо кодове: `PROMO10`, `STUDENT15`, `VIP20`
- Покупка на билети
- Автоматично намаляване на свободните места след покупка
- Отказване на билет и връщане на местата
- Страница „Моите билети“
- Визуализация на електронен билет
- Админ панел за добавяне, редакция и изтриване на събития
- Сравняване на събития
- Сваляне на .ics файл за календар
- Напомняния за избрани събития
- Оценяване със звезди
- Схема на залата в детайлите на събитието

## Стартиране

1. Отвори проекта в IntelliJ IDEA чрез папката, в която се намира `pom.xml`.
2. Избери JDK 17 или 21.
3. Стартирай конфигурацията `Run Eventora (Maven)` или изпълни:

```bash
mvn spring-boot:run
```

4. Отвори в браузър:

```text
http://localhost:8080/visualization
```

## База данни

Проектът използва H2 база по подразбиране. Не е необходима отделна инсталация на MySQL или Docker за локално стартиране. Данните се записват във файл `eventora-db.mv.db`, който се създава автоматично при първо стартиране.

## Бележки

Имейл настройките се подават чрез environment variables. Ако не са зададени, приложението продължава да работи, а кодовете за потвърждение могат да се тестват локално чрез стандартния flow на приложението.

## Последни допълнения

- Основно меню с бърз достъп до събития, любими, билети, кошница и помощ.
- Брояч на закупените билети в навигацията и в страницата „Моите билети“.
- Кратък профилен панел с общ брой билети, поръчки, любими събития и текуща кошница.
- Бързи карти за основните действия в началната страница.
- Обобщение в „Моите билети“: общо билети, брой поръчки и обща сума.


Допълнително:
- Добавена е Eventora Club система с точки при покупка.
- В профила има отделна секция за баланс, ниво и история на точките.
- В кошницата могат да се използват точки за отстъпка.

## Notes
The application uses an embedded H2 database for local testing.
